# AI Interviewer

基于大模型的岗位定制 AI 面试官与面试复盘系统。

## 1. 项目简介

AI Interviewer 是一个面向求职场景的智能面试系统，针对候选人的简历和目标岗位 JD，自动生成贴合岗位的面试问题，并基于候选人的回答进行动态追问、结构化评分和复盘建议输出。

本项目重点解决以下问题：

- 通用面试题库不贴合目标岗位
- 普通 AI 问答缺乏多轮追问能力
- 模拟面试反馈过于笼统，缺少结构化复盘
- 候选人无法快速定位自己的表达短板和项目薄弱点

## 2. 核心能力

- 简历解析：提取候选人的技能、项目经历和潜在追问点
- JD 解析：提取岗位要求、核心能力和重点考察方向
- 面试问题生成：基于简历与 JD 自动生成定制化问题
- 动态追问：根据候选人的回答内容继续深挖技术细节
- 结构化评分：从多个维度对回答质量进行打分
- 面试复盘：输出整体评价、短板分析和改进建议
- 检索增强：基于题库和岗位知识库提升问题相关性和稳定性

## 3. 项目定位

该项目不是一个普通聊天机器人，而是一个面向真实求职场景的结构化 AI 应用系统，强调以下能力：

- 贴岗性
- 多轮面试流程
- 追问逻辑
- 结构化输出
- 工程化落地

## 4. 适用场景

- 候选人进行春招 / 秋招 / 社招模拟面试
- 针对特定岗位进行面试准备
- 对项目经历进行深度复盘
- 训练结构化表达和回答完整性

## 6. 系统架构

系统主要由以下模块组成：

1. Resume Parser（简历解析模块）
2. JD Parser（岗位描述解析模块）
3. Interview Planner（问题生成模块）
4. Follow-up Engine（动态追问模块）
5. Scoring Engine（评分模块）
6. Review Generator（复盘报告模块）
7. RAG Retriever（检索增强模块）
8. Session Manager（会话管理模块）

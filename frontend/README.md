# Frontend

Local Next.js frontend for browser acceptance against the FastAPI backend.

## Run

1. Copy `.env.example` to `.env.local` if the backend is not running at `http://127.0.0.1:8000`.
2. Install dependencies with `npm install`.
3. Start the frontend with `npm run dev`.
4. Start the backend separately, then open `http://127.0.0.1:3000`.
